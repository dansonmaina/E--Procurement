1. HOW TO INSTALL DEMO
 - Extract this archive on your server.
 - Create new database and Install demo tables (demo_database folder):
    The full database is contained on in xcrud_pro.zip
	- xcrud_pro.sql.zip - unzip this folder and restore the database.
 - Configure database connection for xCRUD Pro ( /lib/xcrud/xcrud_config.php ).
 - Try it:
	- http://your_server_url/login.php

2. WHAT FOLDERS IS IN ARCHIVE
 - demo_database - sql files for xcrud pro database
 - pages - all examples is in this folder
 - lib/xcrud - MAIN XCRUD LIBRARY 1.7
